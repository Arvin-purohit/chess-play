# ♟️ Chess Play

A modern chess game built with **Next.js**, **TypeScript**, **Tailwind CSS**, **chess.js**, **react-chessboard**, and **Stockfish AI**.

🌐 **Live Demo:** https://chess-play-two.vercel.app/

---

## ✨ Features

- ♟️ Human vs Human mode
- 🤖 Human vs Computer (Stockfish AI)
- ⏱️ Chess clocks with multiple time controls
- ↩️ Undo move
- 📜 Move history
- ♙ Pawn promotion
- 👑 Check & Checkmate detection
- 🤝 Stalemate detection
- 💀 Draw detection
- 🔄 New Game & Reset Game
- 📱 Responsive design
- 🎨 Clean and modern UI

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Arvin-purohit/chess-play.git
```

Go into the project

```bash
cd chess-play
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
